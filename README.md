# 💼 Explainable Loan Approval System

## 📌 Overview
This project is a machine learning-based system that predicts whether a loan should be approved or not. It focuses on explainability, helping users understand why a decision was made.

---

## 🚀 Features
- Loan approval prediction using ML models  
- Explainable AI (feature importance, insights)  
- Risk analysis of applicants  
- Interactive dashboard using Streamlit  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit
